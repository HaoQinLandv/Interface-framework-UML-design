# 接口框架设计UML图

------

在工作中总是需要对接第三方程序，需要独立出接口模块，并可做如下设计：

> * IFoxItfHandler是超级接口，定义了接口该有的初始化，版本，类型信息
> * DoorCardHandler,PBXHandler是定义具体接口类型的句柄接口
> * AbstractFoxItfHandler抽象类，用来实现初始化IFoxItfHandler，获取接口参数的方法，所有接口句柄都得继承它
